## 📌 Sobre o projeto

Aplicação web que recebe o **peso** e a **altura** do usuário, calcula o **Índice de Massa Corporal (IMC)** e classifica o resultado nas categorias de saúde correspondentes (abaixo do peso, normal, sobrepeso, obesidade).

## ✨ Funcionalidades

- 🧮 Cálculo automático do IMC
- 🏷️ Classificação por faixa de saúde
- 📱 Layout responsivo

## 🛠️ Tecnologias

HTML · CSS · JavaScript

## 🚀 Como executar

```bash
git clone https://github.com/Samuelf27/Calculadora-IMC.git
cd Calculadora-IMC
```

Abra o arquivo **`index.html`** no navegador (ou use a extensão *Live Server* no VS Code).
